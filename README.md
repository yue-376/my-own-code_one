# 轻量级 HIS 课程设计演示程序

这是一个使用 C 语言实现的命令行版医院信息系统（HIS）示例项目，适合作为课程设计演示、功能扩展和基础数据结构练习的起点。

## 当前版本能力

- 分组式菜单：按科室管理、医生管理、患者管理、药品管理、诊疗业务、查询统计、数据导出与样例重置划分主界面。
- 基础数据管理：支持科室、医生、患者、药品的查看、新增、修改、删除和文本文件导入。
- 核心业务：支持患者挂号、住院分配病床、开具处方、药品库存扣减。
- 查询统计：支持查看病房、医疗记录、处方记录、按处方 ID 查询关联医疗记录、综合统计、床位利用率和科室工作量统计。
- 样例数据：程序启动时自动生成 5 个科室、20 名医生、135 名患者（其中 35 名住院患者）和 30 种药品。
- 数据导出：可导出 `patients.txt`、`records.txt`、`prescriptions.txt` 和 `medicines.txt` 供展示或二次处理。

## 代码结构

- `main.c`：程序入口，负责初始化系统和驱动顶层菜单。
- `his.h`：对外暴露的 HIS 接口声明。
- `his.c`：核心数据结构、业务逻辑、查询统计、菜单流程与导入导出实现。
- `Makefile`：提供编译、运行、导出和清理命令。

## 快速开始

```bash
make
make run
```

如果只需要重新导出示例数据：

```bash
make export
```

清理构建产物：

```bash
make clean
```

## 导入文件格式

### 科室导入

```text
name
内科
外科
```

### 医生导入

```text
name,departmentId,title
张三,1,主治医师
李四,2,主任医师
```

### 患者导入

```text
name,age,gender,phone
王五,28,男,13800000001
赵六,32,女,13900000002
```

### 药品导入

```text
genericName,brandName,stock,unitPrice
阿莫西林胶囊,阿莫仙,100,1.80
布洛芬缓释胶囊,芬必得,120,2.50
```

## 说明

当前版本以课程设计演示为目标，重点是链表组织、命令行交互和基础业务流程；如果后续需要进一步贴近真实 HIS，可继续补充分文件细化、持久化存储、权限控制和更完整的病历流程。
